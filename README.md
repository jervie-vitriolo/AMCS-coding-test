# AMCS Group Coding Exam
This solution was based from this problem statement https://github.com/dmitrynovik/jobs-marketplace-api

# Prerequisites 
Must install .NET 10 for ASP.NET Core

# How to run the program
1. Clone and checkout the repo and use the main branch
2. Open the AMCS.sln from the root directory
4. Right click the solution and Restore NuGet Packages
5. Build the solution and Run using IIS Express
6. Swagger UI should appear in the browser https://localhost:44336/swagger/index.html
